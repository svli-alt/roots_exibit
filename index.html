"Roots-Family tree" is a visual project initiated by two visuals artists; Yap Diderot and RaiMe TheGreat. This visual project is representing the family tree through the use of emotions. The project portrays the main common feelings each individuals in a basic family might go through from time to time. This include a set of nine photographs and a series of very short films In which eight are emotions and one is a family tree portrayal.
